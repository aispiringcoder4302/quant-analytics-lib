# VectorBT Pro

Private package for internal use.

## Install

```bash
pip install git+https://github.com/yourusername/vectorbtpro-private.git
```

Or with extras:

```bash
pip install "git+https://github.com/yourusername/vectorbtpro-private.git#egg=vectorbtpro[data,ind,opt,exec]"
```

## Usage

```python
import vectorbtpro as vbt
```
